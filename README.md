# Python scripts
This project contanins my script files I wrote for personal use.

**domain_in_subnet** - performs DNS lookups and determines which subnets the answers belong to.  
**impress_launcher** -  shows LibreOffice Impress slides in a loop.  
**modify_win_shortcuts** - recurcively modifies windows shortcuts' path property in the current direcotry.  
**wkhost** - sends a WoL packet to the specified host.  
**select_subnets** - reads subnets.json and selects subnets for the specified targets.
